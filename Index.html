    Introduction

    Welcome to the Recording API! You can use our API to access Recording API endpoints, which can get information on various Recording files, and folders in our database for your account.

    We have language bindings in Shell, .NET, GO, Java, PHP, Swift and many more! You can view code examples in the dark area to the right, and you can switch the programming language of the examples with the tabs in the top right.

    Implementation

    You need to use the api lib to implement code

    Make sure to replace 553a431e192d2553a431e1930f with your API key.

    Recording uses API keys to allow access to the API. You can register a new Recording API key at the time of verification.

    Recording expects for the API key to be included in all API requests to the server in a header that looks like the following:

    api_key: 553a431e192d2553a431e1930f

    You must replace ApiKey 553a431e192d2553a431e1930f with your personal API key.
